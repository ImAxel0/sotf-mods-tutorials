6.1 - What is SUI?

SUI is an user interface library made my ToniMacaroni to build an UI for mods. Example mods using SUI are Axel's Mod Menu, BuildShare, RedEffect.

Disclaimer

I personally find SUI pretty hard to use, so take all I write with a grain of salt, since not everything could be right or easily explained.
